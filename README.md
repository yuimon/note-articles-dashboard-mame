# まめnote 記事ライブラリ

現役大学生「まめ」（真島さん本人名義note）の実体験note記事を、一覧・検索・カテゴリ絞り込みできる閲覧用ダッシュボードです。

## 概要

- `index.html` … 自己完結型の1ファイルHTML（外部依存なし）。ブラウザで開くだけで動作します。
- 全13記事のタイトル・本文をJS内にJSON埋め込み。
- カテゴリ絞り込み・フリーワード検索（タイトル＋本文）・タイトルクリックで本文展開。
- OSのライト/ダークモード設定に自動対応（フレッシュな青〜緑系の配色）。

## カテゴリ

| カテゴリ | 本数 |
|----------|------|
| AI活用 | 3本 |
| IT就活・資格 | 3本 |
| 投資・トレード | 2本 |
| バイト・お金 | 3本 |
| 習慣・生活 | 2本 |

## 使い方

`index.html` をダブルクリックしてブラウザで開くだけです。ビルド不要・サーバー不要。

## 記事データの更新

記事の元ファイルは `../mame/articles/*.md` にあります。内容を更新した場合は、生成用スクリプトを再実行して `index.html` を作り直してください（カテゴリの割り当ては生成スクリプト内の対応表で管理）。
